## Lo mas dificil del taller

La verdad, lo que mas me costo fue cuadrar bien el responsive para que se viera parecido en desktop, tablet y mobile al mismo tiempo. Uno movia algo en desktop y de una se desajustaba en mobile, entonces toco probar muchas veces, ajustar margenes, tamano de letra y lineas hasta que fuera quedando. Tambien fue enredado lograr que los detalles pequenos (como bordes y espacios) se vieran parecidos a la referencia.
